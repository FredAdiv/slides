# CSS
{id: css}


## CSS - embed in HTML file
{id: css-embed-in-html}

```
<style>
</style>
```

## CSS - include from external file
{id: css-include-from-external-file}

```
<link href="css/style.css" rel="stylesheet">
```

## CSS syntax
{id: css-syntax}

* In CSS syntax looks like this:

```
selector {
   attribute: value;
}
```

## CSS attributes
{id: css-attributes}

```
background-color: red;
color: #AFDE37;
text-align: center;
font-family: verdana;
font-size: 40px;
```

## CSS selectors
{id: css-selectors}

```
html-element {
}

#id {
}

.class {
}
```
